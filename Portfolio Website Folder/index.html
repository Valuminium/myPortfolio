Projects



DermDetect

A machine learning-based model for identifying skin disorders.
The user needs to provide a photo of the infected skin, to see the results.
The model is trained using Artificial Neural Network(ANN) to identify the infected sites on the skin with an accuracy of about 95%.
Achieved an accuracy rate in the model to significantly enhance diagnostic capabilities and reducing misdiagnosis rates by 40%.
